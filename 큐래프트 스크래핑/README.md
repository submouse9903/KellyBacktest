# 큐래프트 카페 학습 아카이브

> **목적**: 네이버 카페 `volanalysis`에 축적된 퀀트 투자, 알고리즘 트레이딩, 자금관리(Kelly Criterion), AI·기술 관련 글들을 체계적으로 수집·정리하고, 그 내용을 깊이 있게 공부하기 위한 개인 학습 프로젝트입니다.
> 
> 이 저장소는 단순히 글을 긁어 모은 공간이 아니라, **2004년~2025년까지 큐래프트 커뮤니티에서 생성된 지식을 하나씩 소화하는 학습 기록 아카이브**입니다.

---

## 1. 왜 이 글들을 공부하는가?

`volanalysis`(큐래프트) 카페는 한국에서 가장 오래된 퀀트 투자·시스템 트레이딩 커뮤니티 중 하나입니다. 이곳에는 다음과 같은 사람들의 글이 쌓여 있습니다.

- **실제 운용을 하는 퀀트 트레이더**
- **수학·공학·물리학 배경을 가진 개발자**
- **자금관리(Kelly Criterion)를 연구하는 학자**
- **AI, 반도체, 금융공학을 연결하려는 기술 인재**

이 글들은 이론서보다 **"실전에서 검증된 아이디어"와 "살아있는 질문"**이 담겨 있어, 단순한 정보 이상의 학습 가치를 지닙니다.

---

## 2. 학습 주제 영역

수집된 60여 개의 글은 크게 5가지 주제로 분류됩니다.

### 2.1 퀀트 투자 & 시스템 트레이딩
- **글로벌 모멘텀 전략** (`systrader79`, `크래프트캐피탈`)
- **프랙탈 지표를 이용한 횡보장 필터링**
- **상한가 따라잡기의 계량적 분석**
- **캔들 차트 패턴 + Machine Learning 분석**
- **Adaptive Kalman Filter / SVM을 활용한 시장 예측**

> **학습 목표**: "감"에 의존하지 않고, 데이터와 알고리즘으로 의사결정을 내리는 구조를 이해한다.

### 2.2 켈리 기준(Kelly Criterion) & 자금관리
- **과학적인 자금관리 전략** (`makilee`, `죽림칠현`, `예전딜러`)
- **풀켈리배팅 vs 하프켈리배팅의 비교**
- **Growth Versus Security in Dynamic Investment Analysis**
- **가치투자와 켈리 공식의 접목**

> **학습 목표**: 수익률을 높이는 것 못지않게, **흐르는 자금을 어떻게 배분하고 보호할 것인가**를 배운다.

### 2.3 퀀트 커리어 & 업계 동향
- **James Simons(륜에상스) 인터뷰 및 상장 기사**
- **Quant Hedge Fund Job Interview Questions**
- **외국계/국내 대형 증권사 Quant Developer, Researcher 채용 정보**
- **파생 모델링 글로벌 프로젝트 구인**

> **학습 목표**: 퀀트 업계가 무엇을 요구하고, 어떤 역량을 중시하는지 파악한다.

### 2.4 금융공학 & 이론
- **Asset Pricing 일반이론**
- **Institutional Order Flow and Stock Returns**
- **Log-Periodic Anti-Bubble**
- **루이 바실리에(Louis Bachelier) — 수리금융의 아버지**

> **학습 목표**: 트레이딩 전략의 밑바탕이 되는 학문적 기초를 쌓는다.

### 2.5 AI·기술 & 미래 산업 (최신 글)
- **AI반도체 기술인재(소프트웨어) 선발대회**
- **알고리즘이 우리의 세계를 어떻게 변화시키는가? (TED)**

> **학습 목표**: 전통적인 퀀트 금융이 AI·반도체 시대와 어떻게 융합되는지 트렌드를 읽는다.

## 3. 데이터 구조 및 활용법

### 원본 데이터 위치
```
큐래프트 스크래핑/output-qraft/
├── post-urls.json          # 수집된 전체 글 목록
├── summary.json            # 성공/실패 요약
├── systrader79/            # 작성자 닉네임 폴터
│   ├── 160617_프랙탈_지표를_이용한_횡보장_필터링_전략/
│   │   ├── content.html    # 본문 HTML (이미지 포함)
│   │   ├── content.txt     # 순수 텍스트 (노트 정리용)
│   │   └── meta.json       # 제목, 날짜, 메뉴, 원문 URL
│   └── ...
└── ...
```

### 정리된 학습 자료 위치
수집된 566개 글의 제목을 분석하여 **시리즈별**과 **연도별**로 재정리한 문서는 아래 폴터에 별도 보관되어 있습니다.

```
정리폰더/
├── 시리즈별_정리/          # 켈리기준, Global Momentum, Python Finance, HFT 등 14개 시리즈
├── 연도별_정리/            # 2004년~2025년 연도별 주제 흐름 분석 (17개 파일)
└── 분석_원본자료/          # 제목 분석용 스크립트 및 원본 데이터
```

### 학습 워크플로우 제안
1. **시리즈별 정리**에서 관심 주제(예: `켈리기준 자금관리.md`)를 열어 전체 흐름과 학습 포인트를 파악한다.
2. **연도별 정리**에서 해당 글이 쓰인 시대적 맥락을 이해한다.
3. 원본 `output-qraft`의 `content.txt`를 열어 핵심 개념을 메모 앱에 옮긴다.
4. 원문 URL을 통해 댓글과 추가 논의를 확인한다.
5. 학습 노트를 작성하고, 이 README에 "학습 기록" 섹션을 계속 업데이트한다.
