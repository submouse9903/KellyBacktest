>>mv모형에서 wE(r)-0.5*namda*w'*sigma*w를 w에 대해 미분하면 E(r)-namda*sigma*w=0을 만족하는 namda는 양변에 w'을 곱하여 정리하면 w'*E(R)/(w'*sigma*w)가 됩니다. 다시 말하면 초과기대수익/분산이 되는데 제 생각에는 namda와 괴델님이 말씀하신 f와 같은 결과를 보이게 됩니다. 켈리공식이 제 기억에는 우위/배당률로 기억하는데 이것이 남다가 되는 과정을 알려주시면 감사하겠습니다 >------------------------------------------------------------------------------------------------------------------------ >  >네..제가 mv 모형에 대해서 정확히 알고있지 못해서 말씀하시는 것을 빨리 알아듣지 못해 죄송합니다. 말씀하신대로 mv 모형과 같은 결과가 나오는 것이 맞습니다. 일단 켈리공식 적용과정에서 위의 결과가 나오는 과정입니다.  >  >우선 켈리공식은 기본적으로 기대성장률 g(f) = E{log[Xn/X0]^(1/n)} 을 최대화시키려는 전략입니다. 여기서 X0는 초기자본, Xn 은 n번의 게임 후의 자본을 나타냅니다. 즉 g(f)는 일종의 복리 수익률의 개념입니다. 우위/배당률 로 알려진 켈리공식은 배당률과 우위가 알려진 기본적인 게임에서 켈리공식을 푼 결과로써 특수한 경우라고 볼 수 있습니다. 이 공식은 다음과 같이 얻어집니다.  >  >게임에서 지면 건 돈을 모두 날리고 게임에서 이기면 건돈 1단위당 b만큼 돈을 딴다고 하면, 매게임 전체자본의 f 비율 만큼을 걸고 S번 이기고 F번 졌다면 n회의 게임 후의 자본은 Xn = X0*(1+bf )^S * (1-f)^F 이 될것입니다. (또한 S+F=n 이 됩니다.) 이때 g(f) 를 계산해보면,  >  >g(f) = E{log[Xn/X0]^(1/n)} = E{S/n * log(1+bf ) + F/n * log(1-f) } >      = p log(1+bf) + q log(1-f) 이 됩니다. ( n이 커지면 S/n 은 이길확률 p, F/n 은 질 확률 q에 수렴합니다. ) >  >이 식을 f에 대해 미분하면, g'(f) = 0 이 되는 f의 값은 (bp-q)/b 가 얻어집니다. 이것이 우위/배당률 공식으로 불리는 것입니다.  >  >켈리공식을 주식시장에 적용하는 하나의 방식은 다음과 같습니다.(The Kelly Criterion in Blackjack, sports betting, and the stock market, Thorp, 1997 의 방식)  >  >주식시장은 위의 게임처럼 확률이 이산적이지 않고 연속적이라서 확률분포를 이용해야만 합니다. 이 경우 성장률 g(f) 는 >  >g(f) = E ln (1 + fX) = ∫ ln (1 + f*x) dP(x) 가 됩니다.  >  >여기서 X는 주식 수익률을 나타내는 확률 변수입니다. f비율만큼 투자하고 나머지는 이자율 r 의 무위험채권에 투자했을때 자본은  >V(f) = V0 * (1 + (1-f)*r + f*X ) 가 됩니다. 따라서  >g(f) = E{ ln( V(f)/V0 ) } = E ln (1 + r + f*(X-r) ) 이 됩니다. >  >수학적인 테크닉으로 X를 없애기 위해서 X를 X를 m+s 와 m-s 의 두 값만을 가질 수 있고 각각의 확률은 0.5인 변수로 가정합니다.  >이렇게 되면 X의 평균은 m, 분산은 s^2 이 되고  >g(f) = E ln ( 1 + r + f*(X-r) ) = 0.5 * ln( 1 + r + f*(m-r+s) ) + 0.5 * ln(1 + r + f*(m-r-s) ) 가 됩니다.  >  >X는 전체 기간 동안의 수익률이기 때문에(예를 들면 1년) 이를 n개의 기간으로 나눕니다. 그러면 m->m/n,  s^2->s^2/n, r->r/n 으로 각각 바뀌게 되고 i 기의 수익률을 나타내는 확률변수 Xi 는 m/n + s*n^(-1/2) 와 m/n - s*n^(-1/2) 두 값을 가질 수 있고 각각의 확률은 0.5 인 확률변수가 됩니다. 그리고 >  >Vn(f)/V0 = ∏ (1 + (1-f)*r + f*Xi ) 가 되어 n->무한대 로 극한을 취하면, >g(f) = E log (Vn(f)/V0) = r + f*(m-r) - s^2 * f^2 /2 가 됩니다. >이를 f 에 대해 미분하면, f* = (m-r)/s^2 >을 얻게 됩니다.  >  >위에서 가정한 주식수익률 확률 변수 X가 단지 두 값만을 가지는 특이한 확률변수이기 때문에 비현실적이다라고 생각하실 수도 있고 그런 생각은 타당합니다. 그러나 다행히도 평균이 m 이고 분산이 s^2 인 어떤 바운드된 확률변수를 X로 잡아도 같은 결과를 가져온다는 것이 증명되어 있습니다.   >  >위에서 얻은 켈리공식의 적용 결과는 말씀하신대로, 람다 = (m-r)/s^2 = 초과기대수익/분산 이라는 결과와 같습니다. >  >  >  >      다음과 같이 켈리공식를 적용 해도 될까요.   m은 이번 거래에서 원하는 기대수익률(목표수익률) 14%   r은 무위험 채권에 투자했을때의 수익률 6%   손절매나 물타기를 통한 최대손실 -7%   99% 확률상에서 3s= 14-(-7) = 21, s=7   최적 켈리 배팅은 (m-r)/s^2= (14-6)/7^2 = 0.16     투표는 표시되지 않습니다.
